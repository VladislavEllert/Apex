### **Динамичный аркадный платформер, вдохновленный классикой Bounce**
Приключения Мячика: BallAdvanture — это современное переосмысление легендарной игры *Bounce*. Управляйте шариком, преодолевайте препятствия и исследуйте детально проработанный пиксельный мир. Собирайте флажки, открывайте проходы и не дайте ловушкам остановить вас!

## 🎮 Об игре
В **Ball Advanture** вы управляете стилизованным мячиком, который должен пройти через запутанные лабиринты футуристического мира. Ваша цель — собрать **все флажки**, тогда **откроется дверь для перехода на новый уровень**, избегайте смертоносных ловушек. 
Забудьте о простом красном шарике. Вы берете на себя роль круглого лесного обитателя. Мы постараемся сохранить основную механику класики, но привнесём существенные изменения.

**Основные особенности:**
- 🔄 **Классический Bounce-геймплей** — прыжки, перекаты и поиск секретных проходов.
- 🌿 **Интерактивное окружение** — платформы, избегайте ловушек.
- 🎭 **Живая анимация персонажа**
- 🎨 **Новый визуальный стиль** — погрузитесь в эстетику Pixel Art с яркими джунглями.
- 🎵 **Атмосферный саундтрек** — электронная музыка, дополняющая визуальный ряд.

### 🛠 Запуск из исходников
1. Установите [Godot 4.4](https://godotengine.org/download/) или [порт для Авроры](https://github.com/savegame/godot/releases/tag/4.4.1-auroraos-4)
2. Клонируйте репозиторий:  
   `git clone https://github.com/top-it-090304/Apex` или скачайте [ZIP](https://github.com/top-it-090304/Apex/archive/refs/heads/main.zip) файл ветки main
4. Откройте папку проекта в редакторе
5. Нажмите `F5` для запуска в режиме отладки
6. Для экспорта: `Проект → Экспорт → Android/Aurora`
