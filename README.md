# Ball Advanture: Приключение мячика

**Ball Advanture: Приключение мячика** — это динамичный аркадный платформер, вдохновленный легендарной игрой *Bounce*. Мы переосмыслили классический геймплей, перенеся его в современную обертку. Управляйте шариком, преодолевайте препятствия и исследуйте детально пиксельный мир.

## 🌌 Описание игры
В **Ball Advanture** вы управляете стилизованным мячиком, который должен пройти через запутанные лабиринты футуристического мира. Ваша цель — собрать все флажки, тогда дверь для перехода на новый уровень будет открыта, избегайте смертоносных ловушек. 
Забудьте о простом красном шарике. Вы берете на себя роль круглого лесного обитателя. Мы постараемся сохранить основную механику класики, но привнесём существенные изменения.

Основные фичи:
- **Классический Bounce-геймплей**: Прыжки, перекаты и поиск секретных проходов.
- **Интерактивное окружение:** Используйте лестницы, платформы и избегайте ловушек.
- **Живая анимация:** Персонаж реагирует на мир — моргает, "дышит" и меняет эмоции.
- **Новый визуальный стиль:** Погрузитесь в эстетику Pixel Art с яркими джунглями и пещерами.
- **Атмосферный саундтрек:** Электронная музыка, идеально дополняющая визуальный ряд.
